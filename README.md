# Padel Points System

A fair, time-based padel scoring system for 4 players on a single court.

## Features
- One-time random shuffle at session start
- Deterministic player rotation
- Unlimited rounds (time-based play)
- Per-player statistics:
  - Rounds played
  - Wins / Losses / Ties
  - Total points
- Simple, mobile-friendly workflow

## Rules
- 2 players play per round
- Total points per round: 16
- Shuffle happens once at the beginning
- Rotation is fixed for the entire session



## Future Improvements
